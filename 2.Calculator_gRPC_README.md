# Calculator gRPC - Project 2

This project demonstrates a calculator service implementation with multiple gRPC operations, input validation, and error handling.

## Project Structure

- **2.Calculator_gRPC_Server**: gRPC server with calculator operations
- **2.Calculator_gRPC_Client**: Interactive console client for calculations
- **Protos/calculator.proto**: Service contract with 4 operations

## Features

- **Four Operations**: Add, Subtract, Multiply, Divide
- **Input Validation**: Handles invalid numbers and operations
- **Error Handling**: Division by zero protection
- **Interactive Client**: Menu-driven console interface
- **Logging**: Server-side operation logging

## Running the Project

### Start the Server
```bash
cd 2.Calculator_gRPC_Server
dotnet run
```
Server will start on https://localhost:7153

### Run the Client
```bash
cd 2.Calculator_gRPC_Client
dotnet run
```

## Operations Available

1. **Add**: Adds two numbers
2. **Subtract**: Subtracts second number from first
3. **Multiply**: Multiplies two numbers
4. **Divide**: Divides first number by second (with zero division check)

## What You Learn

1. **Multiple RPC Methods**: Defining multiple operations in one service
2. **Input Validation**: Handling invalid user input gracefully  
3. **Error Handling**: Implementing business logic validation (division by zero)
4. **Message Structure**: Using success flags and descriptive messages
5. **Interactive Clients**: Building user-friendly console interfaces

## Example Usage

```
Enter operation (1-4) or 'q' to quit: 1
Enter first number: 10
Enter second number: 5
Result: 10 + 5 = 15

Enter operation (1-4) or 'q' to quit: 4
Enter first number: 10
Enter second number: 0
Result: Error: Division by zero is not allowed
Operation failed!
```
