# Hello gRPC - Project 1

This project demonstrates a basic gRPC Hello World service implementation with separate server and client applications.

## Project Structure

- **1.Hello_gRPC_Server**: gRPC server implementation
- **1.Hello_gRPC_Client**: gRPC client implementation
- **Protos/greeter.proto**: Protocol buffer service definition

## Features

- Unary RPC call (SayHello)
- Basic error handling
- Logging on server side
- Interactive client console

## Running the Project

### Start the Server
```bash
cd 1.Hello_gRPC_Server
dotnet run
```
Server will start on https://localhost:5001

### Run the Client
```bash
cd 1.Hello_gRPC_Client
dotnet run
```

## What You Learn

1. **Protocol Buffers**: Define service contracts in .proto files
2. **gRPC Service**: Implement server-side service logic
3. **gRPC Client**: Create and use gRPC clients
4. **Unary RPC**: Simple request-response pattern
5. **Error Handling**: Basic exception handling in gRPC calls
